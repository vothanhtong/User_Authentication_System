# 🔐 Fullstack Authentication System

## 📌 Introduction
This project is a **fullstack authentication system** that provides basic user authentication features including **user registration and login**.

The system is designed to demonstrate how frontend, backend, and database work together to manage users securely.

---

## ⚙️ Features
- User Registration (Sign Up)
- User Login (Sign In)
- Password hashing for security
- JWT authentication
- Input validation and error handling

---

## 🧱 Tech Stack

### Frontend
- ReactJS
- Axios / Fetch API
- TailwindCSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
